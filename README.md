# Park Community School Website

All content uploading and changes to website files are recorded by the IT Department.

A Git repository has been created to manage version control for changes made by the IT Services Department.

---

## Domains

The following domains are associated with Park Community School:

- **parkcommunityschool.co.uk** - Main school website
- **pcs.hants.sch.uk** - Main school website
- **nursery.pcs.hants.sch.uk** - Park Community Nursery

## Project Structure

```
/
├── index.php              # Homepage
├── 404.php                # Error page
├── pages/                 # All subpages
├── partials/              # Reusable components (header, footer, nav)
├── includes/              # PHP utilities and content includes
├── assets/                # Documents, PDFs
├── images/                # Site images
├── css/                   # Stylesheets
│   ├── style.css
│   ├── accessibility.css
│   └── bounce.min.css
└── js/                    # JavaScript (if needed)
```
